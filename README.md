# Quiz Generator (LangChain + Gemini)

## Overview
This project generates beginner-level quiz questions and detailed answers using a two-step LLM pipeline. It demonstrates how to break complex tasks into smaller, reliable components using LangChain.

## Features
- Question generation from a given topic
- Answer generation with explanations
- SequentialChain to automate data flow between steps

## Tech Stack
- Python
- LangChain
- Google Gemini API

## How to Run
1. Create and activate a virtual environment
2. Install dependencies:
   pip install langchain langchain-google-genai
3. Add your API key as an environment variable:
   GOOGLE_API_KEY=your_api_key_here
4. Run the notebook or script

## Notes
- Outputs are cleared for a clean repository
- API keys are not included for security
- Designed as a learning project for LLM workflows
